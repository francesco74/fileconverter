MSG2EML converter

Questo servizio permette di convertire due tipologie di file:
  • una email in formato Outlook (file in formato .msg) in una email che potrai aprire mediante programmi di posta elettronica come Thunderbird (file in formato .eml)
  • un file in formato .pdf in uno in formato .docx quindi permette di rendere editabile un file .pdf se non troppo complesso

Inizia selezionado il file dal tuo computer e premi il tasto "Carica e converti".
Il servizio convertirà il file e ti darà la possibilità di scaricarlo in formato .eml ovvero convertito
